Algoritmo de CLARA

Clara es un algoritmo de partición, se utiliza cuando el conjunto de datos es muy grande (miles de observaciones) con el fin de reducir el tiempo de computación y problemas en el almacenamiento de la RAM.

clara_cluster = clara(CLEAN_P,2)
fviz_cluster(clara_cluster,stand = T,geom = "point",pointsize = 1)+
  theme_bw() +
  labs(title = "RESULTADOS CLUSTERING CLARA")

Validación

fviz_silhouette(clara_cluster)

Un valor de Si cercano a 1 indica que el objeto está bien agrupado, en otras palabras el objeto i es similar a los otros objetos de su grupo, por otro lado, un valor Si cercano a -1 indica que el objeto está mal agrupado y que probablemente si se asignara a otro clúster los resultados mejorarían.

Para finalizar, agreguemos a nuestros datos el vector clustering, para tener un control de en qué clúster quedo cada dato.

Clustering = clara_cluster$clustering
datos = data.frame(cbind(CLEAN_P, Clustering))
